# Grocery_list

An offline-first Grocery Inventory Manager featuring:
- React + Vite Web App
- Express + MySQL Backend
- Flutter Mobile App with offline-first support (sqflite)

## Architecture
- **Web App**: `npm run dev` to start the frontend and backend locally.
- **Mobile App**: Located in the `frontend_mobile` folder. Run `flutter run` to launch.
- **Sync**: Powered by a robust background sync engine with last-write-wins resolution.
